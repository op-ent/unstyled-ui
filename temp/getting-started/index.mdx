# Installation

:::caution

This library is still work in progress and is not ready for production use. Use at your own risk.

:::

To use Unstyled UI in your project, run one of the following commands in your terminal:

<Tabs>
  <TabItem value="npm" label="npm" default>
    ```sh

    npm install @op-ent/unstyled-ui

    ```

  </TabItem>
  <TabItem value="yarn" label="yarn">
        ```sh

    yarn add @op-ent/unstyled-ui

    ```

  </TabItem>
  <TabItem value="pnpm" label="pnpm">
        ```sh

    pnpm add @op-ent/unstyled-ui

    ```

  </TabItem>
</Tabs>
